## Generated artifacts

`release-gate/exports/release-manifest-manifest.json` and `release-gate/reports/*.json` are generated by the release-gate scripts in the private packaging workspace. They are not required to be present in a fresh public clone before running the verifier; `node scripts/verify-package.js` regenerates them locally.

## Exclusion boundary

The catalog must not point to live credentials, OAuth state, private hostnames/IPs, private filesystem paths, chat IDs, raw logs, personal memories/transcripts, vector stores, backups, client/private context, domain-specific private systems, or personal cron jobs.
